Sikulix

  • automates anything you see on the screen of your desktop
  • Windows, Mac or some Linux/Unix
  • image recognition powered by OpenCV
  • no easy access to a GUI's internals
  • OCR powered by Tesseract.

  • Selenium

Installaton

  1. Install Java
  2. Download Sikulix-setup (wget https://launchpad.net/sikuli/sikulix/1.1.2/+download/sikulixsetup-1.1.2.jar)
  3. In Debian 9
    sudo apt-get libopencv-dev wmctrl libtesseract-dev xdotool
  4. java -jar sikulixsetup-1.1.2.jar
  5. Slect pack 1 (which contains Jython)
